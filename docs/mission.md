# Mission

## 비전

**Claude Code 를 쓰는 팀을 위한 Google Analytics.**

팀이 Claude Code 를 얼마나, 어떻게 쓰고 있는지 — 세션·토큰·skill·subagent 활용 — 한 대시보드에서 본다.

## 가치 제안

- **가시성**: 팀 전체의 Claude Code 활동을 한 곳에서. 누가 언제 어떤 skill / subagent 를 썼는가.
- **토큰 한도 관리**: 대부분 구독 플랜이라 "비용" 서사는 약하다. 대신 **토큰 사용/한도** 를 중심에 둔다.
- **Skill / Agent ROI**: 팀이 만든 skill·subagent 중 실제로 쓰이는 것과 죽은 것 구분.
- **0 설정 합류**: `argos` 한 번이면 프로젝트 셋업 + 팀 합류 끝. 이후 팀원은 추가 행동 불필요.

## 타깃

1. **1순위 — Claude Code 를 주력으로 쓰는 개발팀 리드**: 팀의 AI 사용을 이해·코칭하고 싶음.
2. **2순위 — 1인 vibe coder**: 자신의 사용 패턴을 가볍게 확인.
3. **3순위 — 셀프호스트 OSS 사용자**: 자체 인프라에 배포.

## BM

- 현재: 오픈소스 + 관리형 SaaS (무료 티어 + 유료 팀 플랜). 상세 가격/티어는 미정.
- 핵심 과금 지렛대: 팀 멤버 수, 이벤트 보관 기간, 고급 리포트/RBAC.
- 토큰 비용 자체는 사용자의 Claude 구독에서 소모되므로 Argos 는 **관찰 가치**에 과금한다.

## 카피 가이드

- "비용 절감" 같은 문구보다 **"토큰이 어디로 갔는지"** 가 후킹된다 (구독 플랜 사용자 대다수).
- "팀의 AI 사용을 본다" 가 한 줄 요약.
